ReadMe for the primary GitHub repository corresponding to Sherwin et al. 2023,
Single-blind validation of space-based point-source detection and quantification of onshore methane emissions

Key items in this folder include:
-"SatelliteTestingFiguresFullData.ipynb", replication code for almost all analysis in this paper.
-The "Data" folder, which contains the most important data used in this experiment
-"Linear fit simulations 20230114.Rmd", replication code for the supplementary information section comparing different linear fitting specifications.
-The "Raw submissions by operator" folder, containing blinded data as submitted by participating teams.
-"license.md" The license file
-Miscellaneous additional files
